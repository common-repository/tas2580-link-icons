Display an icon after all external links.

== Description ==

The Plugin adds a icon next you your external Links. For known pages the icon get changed on hover with the logo from the site.


== Installation ==

1. Upload `tas2580-link-icons` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
